# Como utilizar

<P>Clone o repositório</P>
<P>Criar banco de dados com nome:</P>

```
bdapi
```

### Instale as dependências

```
$ yarn install
      ou
$ npm install
```
### De os comandos:

```
$ npm run migration:generate
```
```
$ npm run migration:run
```

### Subir o servidor:

```
$ npm start
```
